## Instruções

```bash

git clone https://github.com/pedrowickert123/desafio-ecommerce.git

cd desafio-ecommerce

# instalando dependencias
yarn install

# iniciando o servidor
yarn start

em outro terminal:
# iniciando servidor rest
json-server --port 3333 --watch products.json
```
